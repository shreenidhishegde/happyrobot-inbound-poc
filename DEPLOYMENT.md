### Railway

```bash
# Install Railway CLI
npm install -g @railway/cli

# Login to Railway
railway login

# Initialize project
railway init

# Deploy
railway up
```

**Environment Variables in Railway Dashboard:**
```env
WEBHOOK_API_KEY=your-secret-api-key-here
DATABASE_URL=postgresql://username:password@host:port/database
```
